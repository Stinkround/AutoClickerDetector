
# Autoclick Detection

A 1.21.4 Server-side Paper plugin designed to detect people using an autoclicker



## Authors

- [Stinkround](https://www.github.com/Stinkround)


## Usage/Examples

```java
/amd - Open customization menu
```


## Features

- Click per second detection
- Clicks on a certain interval (ex. a click every 1 second)
- Ban message customization 
- Automatic Banning
